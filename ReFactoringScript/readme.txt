# 해당 폴더는 유니티 프로젝트의 Assets 폴더를 그대로 옮겨 놓은 폴더입니다.

 - LocalData 폴더 : 회원가입에 성공한 계정 정보가 직렬화되어 저장됩니다.

 - Resources 폴더 : Image, Prefab, Animation 등이 저장되어 있습니다.
   ( Resources 폴더 내의 리소스들은 유니티가 게임이 시작되고 끝날때까지 항시 참조합니다. )
   ( 따라서 되도록 다른 방식의 저장 방식을 사용하는 것이 좋다고 합니다. 차후, 시간이 된다면 저장방식이 변경될 수 있습니다. )

 - Scenes 폴더 : 유니티 Scene이 저장되어 있습니다.

 - Script 폴더 : 유니티 내에서 사용될 Script가 저장되어 있습니다.

 - TextMesh Pro : 유니티에서 제공하는 Text UI를 사용하기 위해서 Import한 폴더입니다.

 - Tile Palatte : Tile로 사용할 Palatte에 대한 정보입니다.

 - UnityAssets : 유니티 Assets Store에서 받아온 이미지들이 저장되어 있습니다.
